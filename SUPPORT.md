### Problemas comunes

| Problema | Solución |
|----------|----------|
| `make run` no funciona | Asegúrate de tener JDK 17 y Maven instalados |
| Solo responde un teclado | Haz clic en la ventana del juego para darle foco |
| No se escucha sonido | Revisa que `src/resources/sounds/` tenga los archivos `.wav` |
| `make verify` falla | Ejecuta `mvn -B verify` para ver el error completo |
